# super()

  • super() in Java is used to call the no-arg constructor of parent class.
  • Super must be the first statement in devired class constructor
  • If child class constructor does not invoke super() inside constructor explicitly then Java compiler inserts a call to the super class no-args constructor at compile time.

Example

        class Person {
          Person(){
            System.out.println("Person class Constructor");
          }
        }
 
        class Student extends Person {
          Student(){
            super();
            System.out.println("Student class Constructor");
          }
        }
